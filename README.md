# 网盘图标删除器 Drive Icon Manager

## 介绍

使你不再受‘此电脑’中无用的网盘图标影响

使用管理员权限运行软件会自动读取注册表HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\  并列出子项，选择需要删除的项删除
